# Exercise Timer

A flip clock style countdown timer for your workouts, built with React + TypeScript + Vite.

## Features

- Flip clock UI with smooth card-flip animation
- MM:SS countdown display
- Adjustable duration (set minutes and seconds before starting)
- Start / Pause / Reset controls
- "Done!" indicator when countdown reaches zero

## Getting Started

```bash
pnpm install
pnpm dev
```

Open `http://localhost:5173` in your browser.

## Build

```bash
pnpm build
```

## Stack

- React 19
- TypeScript
- Vite
